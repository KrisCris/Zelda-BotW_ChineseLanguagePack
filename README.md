# Chinese Simplified Language Pack for Zelda: BotW (WiiU / CEMU)

- Using switch ver. translation (使用NS版本的翻译)
- Fixed some errors in the file
- Using `.bnp`  format, better compatibility w/ other mods (使用`.bnp`格式以增加与其他mod的兼容性)
